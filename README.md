# iPhone 15 Pro 3D Website

### 🔗 [Live Demo](https://3d-iphone-phi.vercel.app/)

A premium, interactive 3D website for the iPhone 15 Pro, featuring interactive 3D model animations, high-fidelity video carousels, custom light transitions, and fluid animations.

---

## 🚀 Features

- **Interactive 3D Models**: Explore the iPhone 15 Pro in 3D, customize the colors, and switch sizes (small vs. large) smoothly.
- **Auto-Looping Hero Video**: A beautiful introductory section with an auto-looping teaser video.
- **Dynamic Highlights Section**: Watch the films and features with a custom autoplay and auto-looping video carousel.
- **Premium Animations**: Built with Three.js (React Three Fiber & Drei) and GSAP for responsive, smooth transitions.

---

## 🛠️ Technology Stack

- **Framework**: React 18 + Vite
- **3D Rendering**: Three.js, `@react-three/fiber`, `@react-three/drei`
- **Animations**: GSAP (GreenSock Animation Platform) + `@gsap/react`
- **Styling**: TailwindCSS

---

## 💻 Local Setup & Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/siyam-io/3d-iphone.git
   cd 3d-iphone
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
